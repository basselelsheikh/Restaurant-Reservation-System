To run the program from the jar file, double click on "click_here_to_run_jar.bat".

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

During each run of the program, the program loads the inital data (from the project pdf) from "initialdata.xml".

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

During the runtime of the project, any changes made, for example:  Registering a new user,  Adding a new dish, Customers' orders and tables , etc...  will be saved in "programdata.xml" and "customerdata.xml"
, and when the program is closed, "programdata.xml" and "customerdata.xml" will be cleared.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Additional Features: 

1) The program uses Xml Adapters to be able to marshal and unmarshal childs of abstract superclasses.
2) When a Customer reserves a table, it will be booked and not available for other customers.
3) Registering a new account.
4) Manager can add dishes and tables, and can ban users from the restaurant.
